# Market Live Heatmap

A high-fidelity market heatmap dashboard with real-time global stock data.

## Features
- **Real-Time Data**: Live quotes from Yahoo Finance.
- **Global Coverage**: Monitor US, Europe, Asia-Pacific, and Americas markets.
- **Interactive Heatmap**: Filter by sector and view performance via a color-coded grid.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   node server.js
   ```
3. Open `http://localhost:8007` in your browser.
